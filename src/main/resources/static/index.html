GitHub 회고 MVP

Codaily

GitHub 활동을 모아 회고를 생성하고, 공유 링크와 랭킹 페이지까지 연결하는 Spring Boot 프로젝트다. 기본값은 데모 모드라서 OAuth나 AI 키 없이도 흐름을 바로 확인할 수 있다.

랭킹 페이지 보기 샘플 활동 API 로그인 상태 확인
1단계 백엔드

/api/github, /api/retrospects, /api/rankings 흐름이 준비돼 있다.

OAuth 확장

SPRING_PROFILES_ACTIVE=oauth 와 GitHub 클라이언트 정보를 넣으면 OAuth 로그인으로 확장할 수 있다.

AI 교체 포인트

현재는 템플릿 생성기지만, 서비스 인터페이스가 분리돼 있어 실제 GPT/Claude 호출로 바꾸기 쉽다.
